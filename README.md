# dungeon-crawl

About 3 years ago, I started programming as a hobby, and the first game I ever
made was a simple text-based adventure game called "Dungeon Crawl". I had
forgotten about it for a bit, but after sifting through my existing github
repositories, I found the revised version that I had started around a year ago.
The codebase for that version is contained in [this github
repository](https://github.com/eggsontoast1672/bossfight). I wrote it in java,
and I'm not sure why made that choice. Maybe I was recovering from some sort of
brain-related injury. In all seriousness, java is pretty hard to work with in
vim (my editor of choice) so I'm going back to my roots and migrating over to
python.

This version of Dungeon Crawl is going to be the official rewrite. It will have
new features, active development, and a much cleaner codebase. Suggestions are
welcome, and if you'd like to help me out, you're more than welcome to contact
me on discord at `eggs#5175`.
